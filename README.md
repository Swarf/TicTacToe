# Ultimate Tic Tac Toe

## Game Concept
[Tic Tac Toe to the Next Level](https://en.wikipedia.org/wiki/Ultimate_tic-tac-toe)

## Development

### Run in dev mode:
`npm start`

### Build into ./dist:
`npx webpack`

### Verify dist with:
`python -m http.server 8000 --directory ./dist`
